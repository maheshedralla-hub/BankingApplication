# Banking Application

## Overview
A console-based Banking Application developed using Core Java. It allows users to perform basic banking operations through a menu-driven interface.

## Features
- Show account balance
- Deposit money
- Withdraw money
- Input validation for invalid amounts
- Menu-driven console application

## Technologies Used
- Java
- Core Java
- Scanner Class

## Concepts Used
- Methods
- Loops
- Switch Statement
- Conditional Statements
- User Input Handling

## How to Run

Compile the program:

```bash
javac BankingApplication.java
```

Run the program:

```bash
java BankingApplication
```
